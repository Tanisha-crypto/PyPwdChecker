# Password Strength Checker

A simple yet effective password strength checker built with Python and Tkinter. This application evaluates your passwords based on security criteria and provides real-time feedback.

## Features

- **GUI Interface**: User-friendly graphical interface built with Tkinter
- **Real-time Feedback**: Instant password strength evaluation
- **Color-Coded Results**: 
  - üü¢ Green for Strong passwords
  - üü† Orange for Medium strength passwords
  - üî¥ Red for Weak passwords
- **Security Criteria**:
  - Minimum 8 characters
  - At least one digit (0-9)
  - At least one uppercase letter (A-Z)
  - At least one lowercase letter (a-z)
  - At least one special character (!@#$%^&*...)

## Password Strength Levels

### Strong
- Meets all security criteria
- Password is considered secure

### Medium
- Missing special characters
- Add special characters for better security

### Weak
- Fails one or more security criteria
- Must meet all requirements for a secure password

## Usage

### Running the Application

```bash
python main.py
```

The GUI window will open with the following elements:
- **Password Input Field**: Enter your password (displayed as dots for security)
- **Check Strength Button**: Click to evaluate your password
- **Clear Button**: Reset the fields for a new check
- **Result Display**: Shows the password strength assessment
- **Criteria Display**: Reference guide for password requirements

## How to Use

1. Launch the application using `python main.py`
2. Enter your password in the text field
3. Click the "Check Strength" button
4. View the result:
   - **Green text** = Strong password ‚úì
   - **Orange text** = Medium strength (add special characters)
   - **Red text** = Weak password (needs improvement)
5. Use the "Clear" button to try another password

## Security Notes

- Your password is **never stored** or **transmitted**
- All checks are performed locally on your computer
- The password display is masked with asterisks (*)

## Tips for Strong Passwords

1. Use a mix of uppercase and lowercase letters
2. Include numbers at different positions, not just at the end
3. Add special characters like !@#$%^&*
4. Avoid dictionary words and common phrases
5. Don't use personal information (birthdate, names, etc.)
6. Make it at least 12 characters for better security
7. Use unique passwords for different accounts
